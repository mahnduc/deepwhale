## 🛠 Hướng dẫn cài đặt nhanh

>1. **Tải về:** Nhấp vào nút tải về phiên bản `.msi` ở trên.
2. **Khởi chạy:** Mở file `agent-host_0.1.0_x64_en-US.msi`.
3. **SmartScreen:** Nếu hiện thông báo *"Windows protected your PC"*, hãy chọn **More info** -> **Run anyway**.
4. **Hoàn tất:** Ứng dụng sẽ tự động cài đặt và xuất hiện trên Desktop của bạn.

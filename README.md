## 🚀 Tại sao chọn Uxie?
Khác với các nền tảng SaaS thông thường, Uxie được xây dựng trên triết lý **Local-first**:
*   **Quyền riêng tư tuyệt đối:** Dữ liệu của bạn nằm trên máy của bạn. Chúng tôi sử dụng OPFS (Origin Private File System) để lưu trữ hiệu suất cao ngay trên trình duyệt.
*   **Tốc độ cực nhanh:** Không phụ thuộc vào độ trễ của server.
*   **Offline Ready:** Học tập mọi lúc, mọi nơi ngay cả khi không có internet.

## ✨ Tính năng nổi bật
- [x] **Smart Rich-text Editor:** Trình soạn thảo dựa trên Tiptap tối giản, mạnh mẽ.
- [x] **AI-Powered Quiz:** Tự động tạo câu hỏi trắc nghiệm từ nội dung tài liệu (RAG No-Embedding).
- [x] **Bento Grid Dashboard:** Giao diện hiện đại, trực quan, tối ưu hóa trải nghiệm người dùng.
- [x] **Hybrid Search:** Tìm kiếm ngữ nghĩa kết hợp BM25 giúp truy xuất tài liệu chính xác nhất.
